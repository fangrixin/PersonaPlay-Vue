<h4 align="center">基于SpringBoot+Vue前后端分离的Java快速开发框架</h4>


## 平台简介

  "i人不i"小程序管理后台，通过有趣的 MBTI 测试互动玩法，帮助i人用户与新朋友快速升温，增加彼此了解，找到更多相处建议。

* 前端采用Vue、Element UI。
* 后端采用Spring Boot、Spring Security、Redis & Jwt。
* 权限认证使用Jwt，支持多终端认证系统。
* 支持加载动态权限菜单，多方式轻松权限控制。
